# SMART Command

The `smart` command provides SMART (Self-Monitoring, Analysis, and Reporting Technology) data for a specific disk connected to your Unraid server.

## Usage

```bash
unraid-cli --ip 192.168.1.10 --key YOUR_API_KEY --query smart --disk-id "disk:sda"
```
