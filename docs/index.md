# Unraid API Documentation

Welcome to the official documentation for the Unraid API Python library. This library provides a clean, intuitive interface to Unraid's GraphQL API, enabling developers to programmatically control and monitor Unraid servers.

## Features

- Complete coverage of Unraid GraphQL API endpoints
- Both synchronous and asynchronous client interfaces
- Strongly-typed Pydantic models
- Comprehensive error handling
- API key authentication
- Built-in query caching
- Real-time subscription support
- Command-line interface tool

## About

The Unraid API Python library is designed to make it easy to interact with Unraid servers programmatically. Whether you're building a custom monitoring solution, automating server management tasks, or integrating Unraid with other systems, this library provides the tools you need.
